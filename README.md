# Workspace-Odyssey

## Description

Workspace-Odyssey is a full-stack web application that helps digital nomads find nearby coworking spaces based on a given location. It allows members to search for coworking spaces, leave reviews, view all reviews, and save their favorite spaces to a personal list. The application also includes user authentication using Express sessions, ensuring a secure and personalized experience for each user.


## Features
- **Search for Coworking Spaces:** Users can enter their location to find nearby coworking spaces.
- **User Authentication:** Secure login and registration system powered by Express sessions.
- **Review System:** Members can leave reviews, view all reviews, and rate coworking spaces.
- **Save to Favorites:** Users can save coworking spaces to their personal list for easy access.


## Technologies Used
### Backend:
- **Backend:** Node.js, Express.js
- **Database:** PostgreSQL (with Knex.js for query building)
- **Authentication:** Express Session for user login and session management
- **Location API:** Google Maps API
- **Version Control:** Git, GitHub

### Frontend:
- **Frontend Framework:** React.js
- **Styling:** React-Bootstrap
